# OpenClaw Security Explorer

An interactive reference covering the top 12 security risks and mitigation strategies for [OpenClaw](https://openclaw.ai/) agent deployments.

🔗 **Live site**: [clawsafety.mitbunny.ai](https://clawsafety.mitbunny.ai)

---

## What's inside

- **12 risks** — from prompt injection and malicious skills to exposed gateways and API budget drain
- **12 mitigations** — practical steps to harden your OpenClaw setup
- **Interactive Risk Map** — hover any node to see details, click to highlight connections
- **Bilingual** — full English and Chinese (中文) support
- **No dependencies** — single self-contained HTML file, no build step required

---

## Running locally

Just open `index.html` in any browser — no server needed.

---

## Deploying

The site is deployed via [Vercel](https://vercel.com) from this repo. Every push to `main` auto-deploys.
